# Продажа одежды


Данный программный продукт создан для учета товаров и продаж.



## Начало работы



Вам предоставится архив с программой (7.zip).


### Необходимые условия

Для успешного запуска программного продукта, вам потребуется Microsoft Office Access, который можно скачать с официального сайта Microsoft.


### Установка



Извлечь файлы из архива

Для запуска программы вам необходимо открыть файл "Project1.exe"

Для входа в программу потребуется ввести логин и пароль

Логин: 1234

Пароль: 2021
